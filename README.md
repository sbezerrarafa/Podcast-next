## Prerequisites

- node >=14
- yarn 1.x

## Install

```sh
yarn install
```

## Usage

```sh
yarn start
```

## Run tests

```sh
yarn dev
yarn server

obs: o yarn server é usado, para api- rodar de forma local
```
